Alloco

Description :

L' Alloco est un petit plat typiquement ivoirien à base de banane plantain qui est très consommé dans le pays pour sa qualité et son prix très abordable.

Les ingrédients de l'alloco :

  • banane plantain
  • Huile
  • Sel

Etapes pour faire le plat :

Le plat d'alloco se prépare très facilement :

  1. Découper la banane plantain en petits carré
  2. chauffer de l'huile dans une poèle
  3. Faire frire la banane dans l'huile en rajoutant du sel sur la banane
  4. Enlever après quelques minutes, faire attention, la banane peut très vite bruler
