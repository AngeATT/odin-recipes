Garba

Description :

Le Garba est un petit plat typiquement ivoirien à base de thon et d'attiéké qui est très consommé dans le pays pour sa qualité et son prix très abordable.

Les ingrédients du garba :

  • Thon
  • Attieke
  • Accompagnements : Cube maggi, mayonnaise, oignon et poivre

Etapes pour faire le plat :

Le plat de garba se prépare très difficillement, c'est pourquoi les étapes pour manger le garba sont :

  1. Acheter de l'attiéké
  2. Acheter du poisson thon
  3. Acheter les compléments
  4. Agencer le tout
